# 🍽️ Food Order App (Android)

An Android application built using **Kotlin** that allows users to enter food orders and view a summary.

---

## 🚀 Features

* Take multiple food orders
* Display order summary
* Clean and simple UI
* Color-based order indication

---

## 🛠️ Tech Stack

* Kotlin
* Android Studio
* XML Layouts
